Mission 8


Portefolio

1. Somme


Ecrire une fonction qui prend deux nombres en entrée et retourne au somme

2. Afficher 1..10


Ecrire une fonction qui affiche les nombres de 1 à 10

3. Exos le plus grand entre deux nombre


Ecrire une fonction qui retourne le plus grand entre deux nombres

4. Vérifier si un nombre est pair


Écrire une fonction qui retourne true si un nombre est pair, sinon false.

5. Compter les voyelles dans une chaîne


Écrire une fonction qui compte le nombre de voyelles (a, e, i, o, u) dans une chaîne de caractères donnée.

6. Calculer la factorielle d'un nombre


Écrire une fonction qui retourne la factorielle d'un nombre donné (n!).

7. Vérifier si une chaîne est un palindrome


Écrire une fonction qui vérifie si une chaîne est un palindrome (se lit de la même façon dans les deux sens).

8. Générer une table de multiplication


Écrire une fonction qui retourne la table de multiplication d'un nombre donné.

9. Calculer le carré d'un nombre


Écrire une fonction qui retourne le carré d'un nombre.

10. Trier un tableau de nombres


Écrire une fonction qui trie un tableau d'entiers dans l'ordre croissant.

11. Vérifier la présence d'un élément dans un tableau


Écrire une fonction qui vérifie si un élément donné existe dans un tableau.

12. Concaténer deux tableaux


Écrire une fonction qui combine deux tableaux en un seul.

13. Compter les mots dans une chaîne


Écrire une fonction qui compte le nombre de mots dans une chaîne donnée.

14. Filtrer les nombres pairs dans un tableau


Écrire une fonction qui retourne uniquement les nombres pairs d'un tableau.

15. Calculer la moyenne d'un tableau


Écrire une fonction qui calcule la moyenne d'un tableau de nombres.
